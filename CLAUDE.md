# FlyRank AI Capstone

## Project Overview
This repository is for the FlyRank AI-assisted development capstone.

## Tech Stack
- Node.js
- Git
- GitHub
- Cursor IDE

## Conventions

- Use Conventional Commits for every Git commit.
- Build React components using TypeScript with strongly typed props (avoid using `any`).
- Interactive components must follow WAI-ARIA Authoring Practices.
- Modal dialogs must support keyboard navigation, Escape to close, focus trapping, and return focus when closed.
- Tabs must support Arrow Left and Arrow Right keyboard navigation.
- Disclosure components must use proper `aria-expanded` and `aria-controls` attributes.
- Test every interactive component using keyboard-only navigation before committing changes.
